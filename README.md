
## Getting Started

First, run the fastAPI server:
```bash
python -m api.main
```
Then, run the Next.js server
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result
